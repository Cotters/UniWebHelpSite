What is a box model?

In HTML, all elements can be considered as boxes (unless styled otherwise). The box model can be shown using the image below.

Padding vs Margins: Margins space the content from it's surroundings whilsts padding spaces the content from its border. Here is an example.

Padding: 10px
Margin: 10px

As you can see, the padding creates a space between the border, whilst the margin spaces the content away from surrounding bounderies.
