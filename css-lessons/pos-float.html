Positions & Floats

Positioning

The position property specifies the type of positioning method used for an element.

There are four different position values:

  • static (default)
  • relative
  • fixed
  • absolute

You can position elements top, bottom, left, right, and centre. However, these won't work unless you have first set the position property first, and they work differently depending on the value.

Floats

The float property just specifies whether or not the element should float, and where if it should. The clear property is used to control the behavior of floating elements.
