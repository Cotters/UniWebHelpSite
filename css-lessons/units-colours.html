Units & Colours

CSS Units

CSS supports many types of measurement units. However they can be split into two types:

Absolute units calculate independently of other page content and/or browser defaults.

  • Useful for precise layout
  • Include Pixels (px), Points (pt), Millimetres (mm)

Relative units calculated proportionally against other page content or a browser default.

  • e.g. currently available width, default text size etc.
  • Include Percentages (%), Ems (em), Exes (ex)
  • Also special relative units for text e.g. small, large, x-large etc.

A good design uses both.

CSS Colours

It is important to note that HTML and CSS are American written, so you should always spell colour in the American way, color.

CSS allows rich control over colour. Any colour can be specified using RGB or Hex (hexidecimal) codes. You can sometimes name the colour itself such as p { color: red; } however there are only 17 valid names allowed in CSS2.

An example of HEX colour styling would look like this: p { color: #FF0000; } to produce: Example red text

An example of RGB colour styling would look like this: p { color: rgb(0, 0, 255); } to produce: Example blue text
