Anchor tags

What are anchor tags?

Anchor tags define hyperlinks. This hyperlink references another hypertext document, or sometimes a place on a document. For example the 'Top' button at the bottom of this page links to this page, however the use of JavaScript adds a smooth scroll to the top of the page.

An anchor tag is simply: <a href="#">link</a> which will create something like this: link. The href="#" part of the tag is something called an attribute. Attributes will be looked at in a few lessons time, but in this case the href attribute, which stands for hypertext reference, references the link that the anchor tag will lead to once clicked. Hash (#) simply links to the current document in view, ie this page.

One nice feature to mention is that the text of the link can be anything. I have used this a number of times of different pages of this site. So the following code <a href="www.google.co.uk">Google</a> will link to Google's homepage without the link address being shown. This would produce Google.
