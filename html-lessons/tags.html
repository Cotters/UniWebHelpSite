HTML Tags

What are tags?

Tags have the same format, angle brackets then some abreviation of their function. A paragraph tag looks like this <p></p>. Which is where you might put text like this. This page will give you some insight on what tags do what and when to use them.

Below is a list of popular tags used to create a simple webpage. I have kept the order in which they might appear in your html file.

Tag Description
<!DOCTYPE> Defines the document type
<html> Defines an HTML document
<head> Contains information about the page
<title> Defines document title (appears on browser tab)
<link> Defines a link to external files (stylesheets)
<body> Where the page content is put
<h1> to <h6> Defines different size headings
<p> Defines a paragraph
<a> Defines a hyperlink
<ul> Defines an unordered list
<ol> Defines an ordered list
<li> Defines a list item
<! -- ... --> Defines a comment

There are many more tags, but for simplicity I have kept the list short without leaving anything too important out. Here is a list of alphabetically sorted HTML tags. You might want to bookmark this for future reference as it is extremely helpful to have a list of them all.

Another tag worth mentioning as you will come onto it later is the <script> tag which allows to to add an external JavaScript file to your page, to add functionality like the 'Top' button at the bottom of this page.
