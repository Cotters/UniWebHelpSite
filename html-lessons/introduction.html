What is HTML?

  • HTML stands for Hyper Text Markup Language.
  • Each HTML document is made up of tags.
  • These tags describe different document content.

Structure

Structure is important to consider when writing HTML. Consistant rendering in browsers is something to look out for; this means the code you write will be interpreted correctly in all browser.

Validation is also important when writing web documents. A useful tool to validate your code can be found at https://validator.w3.org/.
